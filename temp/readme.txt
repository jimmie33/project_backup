*_ball.csv is the result for ball tracking.
the format is: <frame number>,<box center x>,<box center y>,<box width>,<box height>


*_head.csv if the result for head tracking.
the format is: <frame number>,<person ID>,<box center x>,<box center y>,<box width>,<box height>

Note that the frame number is the acutal frame index in the ONI file.
